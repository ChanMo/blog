** Usage

Start http server
#+BEGIN_SRC bash
docker run --rm -p 5000:5000 chanmo/pandoc
#+END_SRC

convert to html, use httpie
#+BEGIN_SRC bash
http -f POST :5000/convert/html file@~/demo.docx
#+END_SRC

** APIs

#+BEGIN_SRC bash
/convert/{format}
#+END_SRC
