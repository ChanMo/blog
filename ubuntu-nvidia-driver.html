Install Nvidia Driver On Ubuntu22.04 (Lenovo)

1. Enable SGX

In BIOS, set SGX to Enabled.

2. Install Nvidia Driver

Remove Nvidia Packages 

sudo apt-get remove --purge nvidia-*
sudo apt-get remove --purge '^libnvidia-.*'
sudo apt-get remove --purge '^cuda-.*'

Find Drivers 

ubuntu-drivers devices

DO NOT CHOOSE `nvidia-driver-*-open`

Install 

sudo apt-get install nvidia-driver-525

If there is a dialog about "secure boot", remember your password 

sudo reboot

3. First Reboot

Select "Perform MOK Management", Select "Enroll MOK", Input your password, Select "Ok"

4. Check Driver

nvidia-smi
