1. What is Emacs Org Mode

Org Mode is an authoring tool and a TODO lists manager for GNU Emacs. It relies on a lightweight plain-text markup language used in files with the ‘.org’ extension.

2. Write an article

Use emacs to create a file named "helloworld.org" then use C-c C-e h h to generate a html file.

3. Publish to the web

create a public repository in github, then add your html files, enable to use github page, and your articles are published.

4. Tips

4.1. Add a css file

Edit init.el, add this 

(setq org-html-head "<link rel='stylesheet' href='app.css'>")

4.2. Create a beautiful index.html

You can use orgmode to generate a index.html to list your articles, but the index.html may not be look good as a blog home page.

So you can create a beautiful index.html by using html, css, and javascript.

4.3. SEO

Add this to your orgfile. 

#+TITLE: Hello World
#+KEYWORDS: orgmode, blog
#+DESCRIPTION: a simple blog
